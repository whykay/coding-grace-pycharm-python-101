String formatting

The text enclosed in double/single quotes are strings, they are objects in Python. Each object in Python comes with its own set of in-built functions.
In this example, we will be calling a function, format. This function will look for {} and substitute with whatever it finds inside () in sequential order.
We are giving format the variable, player_name (it can be a number, string and so on), and will be substituted in the first {} it comes across.
