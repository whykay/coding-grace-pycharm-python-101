Lists

A list is a data type in Python. It's a list of objects, e.g. strings, numbers, and other objects.
In this example, a list is created to show the items contained in the treasure chest.
We will also show another way to compare the string the player enters when asked what they would do with the treasure chest.
  • action.lower() - The input from the player will be converted to lower case.
  • If the input from the player matches an of the strings in the list, ["treasure", "chest", "left"], then print the message, "Oooh, treasure!".
    Else ignore it and go and check out the guard instead.
